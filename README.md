# Python Week 1 & 2 Assignments

This repository contains solutions for Week 1 and Week 2 Python assignments.  
The programs demonstrate important Python concepts including:

- Generators  
- Iterators  
- List Comprehension  
- Tuples  
- Object-Oriented Programming (OOP)  
- File Handling  
- Pickle Module  
- NumPy  
- Matplotlib  

All programs take input from the user through the command line.

## Problems Implemented

### Week 1
1. Generate Fibonacci numbers using a Generator  
2. Reverse list using custom Iterator class  
3. Prime numbers between 0–10 using list comprehension  
4. Convert list to list of tuples (index, value)

### Week 2
5. Library management system using OOP and pickle  
6. Matrix diagonal swap using NumPy  
7. Text file processing (capitalize words + count spaces)  
8. Random number frequency analysis with NumPy & Matplotlib

## Python Version
Python 3.11

## Libraries Used
- pickle  
- numpy  
- matplotlib
